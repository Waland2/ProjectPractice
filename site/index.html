Synthadry
Главная О проекте Участники Журнал Ресурсы

Группа проектов игровой индустрии. Проект «Synthadry»

Цель Создание 3д игры с динамичным сюжетом, классными игровыми механиками, и вывод на любую платформу для возможности скачивания
Ключевые задачи Развитие и обучение членов команды, создание необходимых игровых механик, создание крутого дизайна, реклама для продвижения игры, вывод на любую платформу для возможности скачивания
